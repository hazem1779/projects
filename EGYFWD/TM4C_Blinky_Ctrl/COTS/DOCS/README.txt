NOTES: 

1- This project uses the General Purpose Timer PWM mode and it uses interrupts to handle both SW1 and SW2 to obtain the seconds wanted by the user 
in which the program should blink the led on and off in the infinite loop

2- Project is tested using TIVA C tm4c123gh6pm board 

3- All documents are provided in Docs folder

4- Project files are integerated from App,Inc,Src,Cfg folders
